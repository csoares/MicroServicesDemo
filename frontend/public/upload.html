Handled by Node.js + Express Gallery Service → RabbitMQ → Python + FastAPI Image Processor

📷

Drag & drop an image here, or

JPEG, PNG, WebP — max 10 MB

Event flow after upload:
  1. gallery-service saves original to /uploads/originals/
  2. gallery-service publishes photo.uploaded to RabbitMQ
  3. image-processor consumes event, resizes with Pillow, saves thumbnail + medium
  4. image-processor publishes image.processed to RabbitMQ
  5. gallery-service updates the database row — status: pending → processed
  6. Gallery page polls and shows the thumbnail

Watch the RabbitMQ Management UI (guest/guest) while uploading!
